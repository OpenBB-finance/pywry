# Tauri Web Viewer

To use just run the following:
- `pip install maturin`
- `maturin develop`


Basic Usage:
```python
>>> import pywry
>>> pywry.show_html("<h1>Welcome to WRY with python</h1>")

```

Note: There is currently an issue if you try to run this inside an X86_64 conda
environment on an M1 machine.
